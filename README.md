⚙️ Installation
Clone Repository
git clone https://github.com/yourusername/AI-PDF-Teacher.git

cd AI-PDF-Teacher
Install Dependencies
pip install -r requirements.txt
Add Gemini API Key
Streamlit Secrets
GEMINI_API_KEY="YOUR_API_KEY"

or

export GEMINI_API_KEY="YOUR_API_KEY"
▶️ Run Application
streamlit run app.py
